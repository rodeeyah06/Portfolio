# Portfolio Website

## Overview
This project is a responsive personal portfolio website created to showcase professional skills, projects, and experience in a clean and accessible way. It serves as a digital profile that highlights technical abilities and personal branding.

The portfolio was built with a focus on simplicity, clarity, and usability.

---

## Features
- Responsive design for desktop and mobile devices  
- Clean and minimal UI  
- Sections for personal information, skills, and projects  
- Interactive elements powered by JavaScript  

---

## Tech Stack
- **HTML5** – Structure and content  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and dynamic behavior
